# 1、简介

>layui（谐音：类 UI) 是一套开源的 Web UI 解决方案，采用自身经典的模块化规范，并遵循原生 HTML/CSS/JS 的开发方式。
>
>layui 区别于那些基于 MVVM 底层的前端框架，它更多是面向后端开发者。

