Dao 层接口调用使用方式

（1）调用继承的接口中的方法

（2）引入jpql（jpa查询语言）语句进行查询，sql操作的是数据表和字段，jpql操作的是对象和属性

（3）引入原生的sql语句

（4）按照方法命名规则查询

```
// 比如：按照 name 和 address 模糊查询（like）
// 方法名以findBy开头
// 属性名（首字母大写）
// 查询方式（模糊查询、等价查询），如果不写查询方式，默认等价查询
```
